# PVDasm x86 (32-bit) Opcode Coverage

This document lists all instruction sets and opcodes supported by PVDasm when disassembling 32-bit PE executables. Every entry below has been verified against the source code.

## x87 FPU (0xD8–0xDF)

Full coverage of all ESC 0–7 floating-point instructions.

### 0xD8 — ESC 0
- **Memory:** FADD, FMUL, FCOM, FCOMP, FSUB, FSUBR, FDIV, FDIVR (all m32fp)
- **Register:** FADD ST,ST(i) · FMUL ST,ST(i) · FCOM ST(i) · FCOMP ST(i) · FSUB ST,ST(i) · FSUBR ST,ST(i) · FDIV ST,ST(i) · FDIVR ST,ST(i)

### 0xD9 — ESC 1
- **Memory:** FLD m32fp, FST m32fp, FSTP m32fp, FLDENV, FLDCW, FSTENV / FNSTENV, FSTCW / FNSTCW
- **Register:** FLD ST(i), FXCH ST(i), FNOP
- **Special (0xD9 E0–FF):** FCHS, FABS, FTST, FXAM, FLD1, FLDL2T, FLDL2E, FLDPI, FLDLG2, FLDLN2, FLDZ, F2XM1, FYL2X, FPTAN, FPATAN, FXTRACT, FPREM1, FDECSTP, FINCSTP, FPREM, FYL2XP1, FSQRT, FSINCOS, FRNDINT, FSCALE, FSIN, FCOS

### 0xDA — ESC 2
- **Memory:** FIADD, FIMUL, FICOM, FICOMP, FISUB, FISUBR, FIDIV, FIDIVR (all m32int)
- **Register:** FCMOVB, FCMOVE, FCMOVBE, FCMOVU, FUCOMPP

### 0xDB — ESC 3
- **Memory:** FILD m32int, FIST m32int, FISTP m32int, FLD m80fp, FSTP m80fp
- **Register:** FCMOVNB, FCMOVNE, FCMOVNBE, FCMOVNU, FCLEX / FNCLEX, FINIT / FNINIT, FDISI, FENI, FUCOMI, FCOMI

### 0xDC — ESC 4
- **Memory:** FADD, FMUL, FCOM, FCOMP, FSUB, FSUBR, FDIV, FDIVR (all m64fp)
- **Register:** FADD ST(i),ST · FMUL ST(i),ST · FSUBR ST(i),ST · FSUB ST(i),ST · FDIVR ST(i),ST · FDIV ST(i),ST

### 0xDD — ESC 5
- **Memory:** FLD m64fp, FST m64fp, FSTP m64fp, FRSTOR, FSAVE / FNSAVE, FNSTSW m16
- **Register:** FFREE ST(i), FFREEP ST(i), FST ST(i), FSTP ST(i), FUCOM ST(i), FUCOMP ST(i)

### 0xDE — ESC 6
- **Memory:** FIADD, FIMUL, FICOM, FICOMP, FISUB, FISUBR, FIDIV, FIDIVR (all m16int)
- **Register:** FADDP ST(i),ST · FMULP ST(i),ST · FCOMPP · FSUBRP ST(i),ST · FSUBP ST(i),ST · FDIVRP ST(i),ST · FDIVP ST(i),ST

### 0xDF — ESC 7
- **Memory:** FILD m16int, FIST m16int, FISTP m16int, FBLD m80bcd, FILD m64int, FBSTP m80bcd, FISTP m64int
- **Register:** FNSTSW AX, FUCOMIP ST,ST(i), FCOMIP ST,ST(i)

## EVEX / AVX-512 (Partial)

EVEX prefix decoding is supported with the following features:

### EVEX Prefix Fields
- **R, R', X, B bits:** Extended register encoding (ZMM0–ZMM31)
- **W bit:** Operand size (32/64-bit element)
- **VVVV + V':** Source register specifier (5 bits, up to 32 registers)
- **aaa:** Opmask register (k0–k7)
- **z bit:** Zeroing-masking vs. merge-masking
- **b bit:** Embedded broadcast (1to16, 1to8, 1to4, 1to2)
- **LL:** Vector length (00=128, 01=256, 10=512)

| **BMI2** | BZHI, PDEP, PEXT, MULX, SARX, SHLX, SHRX (VEX 0F38 F5-F7), RORX (VEX 0F3A F0) |
| **F16C** | VCVTPH2PS (VEX 0F38 13), VCVTPS2PH (VEX 0F3A 1D) |
| **KADD** | KADDB/W/D/Q (VEX 0F 4A) |

## Known Gaps (Not Implemented)

The following instruction set extensions are **not** currently supported:

| Extension | Key Instructions |
|-----------|-----------------|
| **TSX (extended)** | XBEGIN (C7 F8 rel32), XABORT (C6 F8 imm8) |
| **CET (extended)** | RDSSPD/Q, INCSSPD/Q |
| **AVX-512 DQ** | VPMULLQ, VCVTQQ2PD, VPMOVM2D/Q, etc. |
| **AVX-512 BW** | VPMOVWB, VPMOVSWB, VPACKSSWB zmm, etc. (partial: VPSRLVW/VPSRAVW/VPSLLVW and VDBPSADBW are implemented in EVEX) |
| **AVX-512 VL** | 128/256-bit EVEX forms of AVX-512 instructions |
| **AVX-512 IFMA** | VPMADD52LUQ, VPMADD52HUQ |
| **AVX-512 VBMI** | VPERMB, VPERMI2B, VPERMT2B, VPMULTISHIFTQB |
| **GFNI** | GF2P8MULB, GF2P8AFFINEQB, GF2P8AFFINEINVQB |
| **VAES/VPCLMULQDQ** | Vectorized AES and carry-less multiply |
